MySQL安装

安装完成MySQL，需要在系统的偏好设置启动MySQL服务。

设置MySQL命令别名：

alias mysql=/usr/local/mysql/bin/mysql
//设置MySQL的密码
alias mysqladmin=/usr/local/mysql/bin/mysqladmin

设置MySQL的密码

//初次安装MySQL需要设置密码
mysqladmin -u root password ***(#初始密码)

//修改密码
mysqladmin -u root -p password new***(#最新密码)

连接数据库

mysql -u root -p
如果是登录远程主机上的mysql数据库
mysql -h 主机地址 -u 用户名 -p 用户密码

设置MySQL的编码方式：

在Mac或Linux上，需要编辑MySQL的配置文件，把数据库默认的编码全部改为UTF-8。MySQL的配置文件默认存放在/etc/my.cnf或者/etc/mysql/my.cnf：

[client]
default-character-set = utf8

[mysqld]
default-storage-engine = INNODB
character-set-server = utf8
collation-server = utf8_general_ci

重启MySQL后，可以通过MySQL的客户端命令行检查编码：

redis安装

~ wget http://download.redis.io/releases/redis-3.0.5.tar.gz 
~ tar xzf redis-3.0.5.tar.gz
~ mv redis-3.0.5 redis
~ cd redis
~ make
~ make test
~ make install

编辑redis.config文件指定数据文件保存的路劲

dir /Users/ZhiHua/redis_config/redis_data/

启动redis服务

redis-server /Users/ZhiHua/redis_config/redis.conf

设置redis开机启动

~ sudo vi /Library/LaunchDaemons/io.redis.redis-server.plist

文件内容：

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple Computer//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>io.redis.redis-server</string>
  <key>ProgramArguments</key>
  <array>
        <string>/usr/local/bin/redis-server</string>
        <string>/usr/local/etc/redis.conf</string>
  </array>
  <key>RunAtLoad</key><true/> </dict>
</plist>

使用launchctl设置开机自动启动

~ sudo launchctl load /Library/LaunchDaemons/io.redis.redis-server.plist

使用launchctl启动redis server

~ sudo launchctl start io.redis.redis-server

使用launchctl停止redis server

~ sudo launchctl stop io.redis.redis-server

注意点：

  1. 调用[self setNeedsDisplay];可以重新绘制图形。

画线

- (void)drawRect:(CGRect)rect {

    //获取图形上下文，图形上下文输出的目标是self-》也就是当前的试图对象。
    CGContextRef context = UIGraphicsGetCurrentContext();

    //设置线条的颜色
    [[UIColor yellowColor] set];

    //设置线宽
    CGContextSetLineWidth(context, 4);

    // 设置线的头尾的样式
    CGContextSetLineCap(context, kCGLineCapButt);

    //连接点的样式
    CGContextSetLineJoin(context, kCGLineJoinRound);

    //线条的起点
    CGContextMoveToPoint(context, 5, 5);
    //线条的终点
    CGContextAddLineToPoint(context, CGRectGetWidth(rect), CGRectGetHeight(rect));
    //渲染线条
    CGContextStrokePath(context);
}

画三角形

原理：在画线的基础上添加多个点来连成连续的线，将路径关闭起来CGContextClosePath(context);就可以形成三角形。

画矩形

方法一：通过画四个点连成三条线，然后将起点和终点连接起来。

方法二：

- (void)drawRect:(CGRect)rect {

    CGContextAddRect(context, CGRectMake(100, 100, 50, 50));

    //设置线条的颜色
    CGContextSetRGBFillColor(context, 0.2, 0.8, 0.2, 1);

//    [[UIColor yellowColor] setFill];

//    //渲染-》【空心】
//    CGContextStrokePath(context);
    //渲染-》【实心】

画圆、弧形、扇形

//画弧形
- (void)drawRect:(CGRect)rect {

    CGContextRef context = UIGraphicsGetCurrentContext();

    /*!
     *  @author zhihua, 16-06-02 17:06:03
     *  圆心的点
     *  半径
     *  起始的弧度
     *  结束的弧度
     *  clockwise 0 顺针 1 逆时针
     *  @brief
     */
    CGContextAddArc(context, CGRectGetMidX(rect), CGRectGetMidY(rect), 100, 0, M_PI*0.65, 1);

    CGContextStrokePath(context);

}

//画扇形
- (void)drawRect:(CGRect)rect {

    //获取图形上下文
    CGContextRef context = UIGraphicsGetCurrentContext();

    CGContextMoveToPoint(context, CGRectGetMidX(rect), CGRectGetMidY(rect));
    CGContextAddArc(context, CGRectGetMidX(rect), CGRectGetMidY(rect), 50, 0, M_PI*0.25, 1);

    CGContextClosePath(context);

    CGContextStrokePath(context);
}

//画弧形
- (void)drawRect:(CGRect)rect {

    //获取图形上下文
    CGContextRef context = UIGraphicsGetCurrentContext();

    CGContextAddEllipseInRect(context, CGRectMake(CGRectGetMidX(rect), CGRectGetMidY(rect), 50, 50));

    CGContextStrokePath(context);

}

IBInspectable

因为可检查属性仅仅是用户定义的运行时属性顶部的接口，所以支持相同的类型列表：布尔，字符串和数字（即，NSNumber 或任何数值类型），以及 CGPoint、CGSize、CGRect、UIColor 和 NSRange，额外增加了 UIImage。

IBDesignable

代码:

import UIKit

@IBDesignable
class VisiableView: UIView {

    @IBInspectable var cornerRadius:CGFloat = 0.0 {
        didSet {
            layer.cornerRadius = cornerRadius
        }
    }

    @IBInspectable var masksToBounds:Bool = false {
        didSet {
            layer.masksToBounds = masksToBounds
        }
    }
}

在OC中的实用方法基本类似。

prepareForInterfaceBuilder()：此方法与你代码的其余部分一起编译，但只有当视图正在准备在 Interface Builder 显示时执行。

#if !TARGET_INTERFACE_BUILDER
    // this code will run in the app itself
#else
    // this code will execute only in IB
#endif
