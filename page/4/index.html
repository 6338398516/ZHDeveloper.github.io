Swift闭包

发表于 2016-05-09   |   分类于 Swift语法   |     |   阅读次数

闭包:

  1. 函数是闭包的一种特殊。
  2. 闭包表达式（匿名闭包）- 能捕获上下文中闭包的值。

闭包的语法:

  1. in关键字是便于区分返回值和执行语句。
  2. 闭包表达式的类型和函数的类型一样，是参数上加返回值。也就是in之前的部分。
  3. 如果闭包是最后一个参数, 可以直接将闭包写到参数列表后面, 这样可以提高阅读性. 称之为尾随闭包

eg:

//完整写法
let say:(String)->String = {
    (name:String) -> String in
    return "hello \(name)"
}

//没有返回值的写法
let say:(String)->Void = {
    (name:String) in
    print("hello \(name)")
}

//无参无返回值
let say = {
    print("hello")
}

func bubbleSort(inout array:[Int], cmp: (Int, Int) -> Int)
{
    let count = array.count;
    for var i = 1; i < count; i++
    {
        for var j = 0; j < (count - i); j++
        {
            if cmp(array[j], array[j + 1]) == -1
            {
                let temp = array[j]
                array[j] = array[j + 1]
                array[j + 1] = temp
            }
        }
    }
}

// 闭包作为参数传递
bubbleSort(&arr, cmp: {
    (a: Int, b: Int) -> Int in
    if a > b{
        return 1;
    }else if a < b
    {
        return -1;
    }else
    {
        return 0;
    }
})

闭包表达式的优化:

  1. 类型优化, 由于函数中已经声明了闭包参数的类型, 所以传入的实参可以不用写类型
  2. 返回值优化, 同理由于函数中已经声明了闭包的返回值类型,所以传入的实参可以不用写类型
  3. 参数优化, swift可以使用$索引的方式来访问闭包的参数, 默认从0开始
  4. 如果只有一条语句可以省略return

eg:

bubbleSort(&arr){
//    (a , b) -> Int in
//    (a , b) in
    if $0 > $1{
        return 1;
    }else if $0 < $1
    {
        return -1;
    }else
    {
        return 0;
    }
}

Swift函数

发表于 2016-05-06   |   分类于 Swift语法   |     |   阅读次数

函数:完成特定功能的代码块。给函数定义一个有意义的名字，在合适的时候调用。

定义:
func 函数名称(参数名:参数类型, 参数名:参数类型…) -> 函数返回值 {函数实现部分}

//无返回值
func say() -> Void{
    print("hello")
}

func say() {
    print("hello")
}

//有参无返回值
func sayWith(name:String) {
    print("hello \(name)")
}

//无参有返回值
func personInfo() -> (String,Int) {
    return ("zhangsan",23)
}

//有参有返回值
func info(name:String, age:Int) -> String
{
    return "name = \(name), age = \(age)"
}

//函数嵌套
func bubbleSort(inout array:[Int]) -> Void {

    //定义一个交换函数
    func swap(inout a:Int,inout b:Int)
    {
        let temp = a
        a = b
        b = temp
    }

    let count = array.count

    for i in 1..<count {
        for j in 0 ..< (count-i) {
            if array[j]>array[j+1]
            {
                swap(&array[j], b: &array[j+1])
            }
        }
    }

}

//要使用inout必须是变量，因为inout会改变值
var array = [54,43,6,63,23,1,52]
bubbleSort(&array)

函数参数

  1. 函数的第一个参数默认是内部参数，之后的参数即是内部参数也是外部参数。
  2. 自定义外部参数，在内部参数前加外部参数和空格
  3. Swift2.0过时,在参数前面加上#相当于该参数即是内部参数, 也是外部参数
  4. 参数默认值，在定义函数的时候给某个参数赋值, 当外部调用没有传递该参数时会自动使用默认值
  5. 在其它语言中默认参数智能出现在参数列表的最后面, 但是在Swift中可以出现在任何位置
  6. inout参数, 如果想在函数中修改外界传入的参数, 传递参数本身而不是参数的值
  7. …表示函数的参数有多个，相当于数组。变参必须放到函数列表的最后。

eg:

dividend和divisor都是外部参数，a和b是内部参数
func divisionOpertaion(dividend a: Double, divisor b:Double) -> Double{
    return a / b
}

//参数默认值的使用
func calculateWith(number1 a:Int,andNumber b:Int = 10) -> Int {
    return a+b
}

let a = calculateWith(number1: 8)

函数类型

函数的类型是由参数类型和返回值的类型组成。
可以使用函数类型来定义常亮和变量。

//函数的类型为(Double,Double)->Double
func divisionOpertaion(dividend a: Double, divisor b:Double) -> Double{
    return a / b
}

//函数类型作为参数，在函数内部调用
func calFunc(a: Int, b: Int ,Option:(Int, Int) -> Int) ->Int
{
    return Option(a, b)
}

//函数类型作为返回值
func max(a:Int,b:Int) -> Int {
    return a>b ? a:b
}

func min(a:Int,b:Int) -> Int {
    return a<b ? a:b
}

func getMax(getMax:Bool) -> (Int,Int)->Int {
    return getMax ? max : min
}

switch

特点：

  1. 默认不可以穿透，使用关键字fallthrough使语句穿透
  2. 可以不写break
  3. 必须有default
  4. 在case中定义变量可以不用加大括号
  5. 可以和区间和元组匹配
  6. switch的结构相对于if(){}else if(){}else{}

eg.

var rank1 = "A"
switch rank1{
    case "A", "B": // 注意OC不能这样写
        print("优")
    case "C":
        print("差")
    default:
        print("没有评级")
}
//两个实现的效果是一样的
switch (rank) {
    case "A":
        fallthrough
    case "B":
        NSLog("优");
    case "C":
        NSLog("差");
    default:
        NSLog("没有评级");
}

//与区间匹配
let aNum = 10

switch aNum {
    case 0...10:
        print(0)
    case 10...20:
        print(1)
    case 20...30:
        print(2)
    case 30...40:
        print(3)
    default:
        print(aNum)
}
//区间与元组搭配
let aNum = (89,97)

switch aNum {
    case (0..<60,0..<60):
        print("不及格")
    case (60..<70,60..<70):
        print("差")
    case (70..<80,60..<80):
        print("良")
    case (80...100,80...100):
        print("优秀")
    default:
        print(aNum)
}

//值绑定
var point = (1, 10)
switch point{
    case (var x, 10): // 会将point中X的值赋值给X
        print("x= \(x)")
    case (var x, var y): // 会将point中XY的值赋值给XY
        print("x= \(x) y= \(y)")
    case var( x, y):
        print("x= \(x) y= \(y)")
    default:
        print("Other")
}

//根据条件绑定
var point = (100, 10)
switch point{
    // 只有where后面的条件表达式为真才赋值并执行case后的语句
    case var(x, y) where x > y: 
        print("x= \(x) y= \(y)")
    default:
        print("Other")
}

常亮、变量

常量：使用let来声明的变量，常亮的值一旦确定就不可再改变，并且常亮要在初始化时进行赋值，否则报错。
变量：使用var来声明的变量，变量的值可以多次改变。

1.先定义后初始化
var num: Int
num = 3

注：如果声明变量时没有指定数据类型会报错：没有指定数据类型（type annotation missing in pattern）。在Swift中定义一个变量，以后使用再初始化必须在定义时指定变量的类型（类型标注）

2.定义的同时初始化
var num: Int = 3
var num = 3

注：在Swift中变量的定义的同时进行初始化，可以不指定变量的类型。因为在编译期间会根据初始化的值来自动推断变量的类型。以后在开发中定义的同时初始化变量，可以不用写变量的类型，除非需要明确数据类型的长度。

常亮、变量命名：不能包含数学符号，箭头，保留的（或者非法的）Unicode 码位，连线与制表符。也不能以数字开头，但是可以在常量与变量名的其他地方包含数字。

错误：
var 3x = 10
var x-+3 = 10

正确：
var 🐶 = 10
var 哈哈 = 20

Swift:var intValue:UInt = -10 会报错
//溢出
OC:int intValue = INT_MAX + 1; 不会报错
Swift:var intValue:UInt = UInt.max + 1 会报错

数据类型的相互赋值(隐式类型转换)
OC可以
int intValue = 10;
double doubleValue = intValue;

Swift:不可以
var intValue:Int = 10
var doubleValue:Double = intValue
在Swift中“值永远不会被隐式转换为其他类型”(OC中可以隐式类型转换), 以上语句会报错

类型转换

Swift中是不会进行隐式类型转换，但可以进行显示类型转换。

var intValue: Int = 5
var doubleValue: Double
doubleValue = Double(intValue)
注意:Double()并不会修改intValue的值, 而是通过intValue的值生成一个临时的值赋值给doubleValue

Bool类型

let isOpen = true;
let isOpen = 1; //会报错

Swift中的if的条件只能是一个BOOL值或者返回BOOL的表达式。

元组

元组：在其他语言中早就有元组的概念，但相对于OC的开发者来说这是一个新的概念。将多个类型相同或者不同的类型的值用括号括起来表示元组。

let score = ("张三",90,89.4)
print(score)
print(score.0)
print(score.1)
print(score.2)

元组其实和结构体很像，只是不需要提前定义类型。元组其实是复合类型，小括号中可以写任何类型。

let score: (String,Double,Double) = ("张三",90,89.4)

元组的其他定义方式：指明元组元素的名称。

let score = (s_name:"张三",math_score:90.0,chinese_score:89.4)
print(score)
print(score.s_name)
print(score.math_score)
print(score.chinese_score)

通过指定名称提取元组对应的值，对应位置的值赋予对应位置的名称。

let (s_name,math_score,chinese_score) = ("张三",90.0,89.4)
print(s_name)
print(chinese_score)

如果不关心元组中的某个值，可以使用_忽略提取。

let (s_name,math_score,_) = ("张三",90.0,89.4)

可选值

可选值optionals有两种状态：

  1. 有值
  2. 没有值，没有值就是nil

eg:

// 有值
var optionValue: Int? = 9

// 没值
var optionValue1: Int?
var optionValue2: Int? = nil

可选值可以使用if来判断

if optionValue != nil
{
    print("有值")
}
else
{
    print("没值")
}

提取可选类型的值（强制解包）

会将optValue中的整型值强制拿出来赋值给变量result, 换句话说就是告诉编译器optValue一定有值, 因为可选类型有两种状态有值和没有值, 所以需要告诉编译器到底有没有值

需要注意的是如果强制解析optValue, 但是optValue中没有值时会引发一个运行时错误

var optValue5: Int? = 9
var result1: Int = optValue5!
print(result1)

//报错:
var optValue6: Int?
var result2: Int = optValue6!
print(result2)

可选绑定：
为了更安全的解析可选类型的值，一般情况下使用可选绑定。
如果optValue没有值是不会进行任何操作，如果optionValue有值就会返回true并将optionValue的值赋予result,然后执行大括号中的内容。

if let result = optionValue
{
    print(result)
}

字符和字符串

字符：

  1. Swift的字符是使用双引号
  2. Swift中的字符类型和OC中的也不一样, OC中的字符占一个字节,因为它只包含ASCII表中的字符, 而Swift中的字符除了可以存储ASCII表中的字符还可以存储unicode字符

eg:

var chara: Character = "a"
//双引号中只能放一个字符
var chara: Character = "ab"        //错误

字符串：

字符是单个字符的集合，字符串是多个字符的集合，想要存放多个字符，使用字符串。

var str: String = "Product"


C语言中的字符串是以\0结尾的, 例如:
char *stringValue = "abc\0bcd";
printf("%s", stringValue);
打印结果为abc

OC语言中的字符串也是以\0结尾的, 例如:
NSString *stringValue = @"abc\0bcd";
NSLog(@"%@", stringValue);
打印结果为abc

var stringValue2 = "abc\0bcd"
print(stringValue2)
// 打印结果为abcbcd
// 从此可以看出Swift中的字符串和C语言/OC语言中的字符串是不一样的

字符串常用的方法

计算字符串的长度

/*
计算字符串长度:
C:
char *stringValue = "abc李";
printf("%tu", strlen(stringValue));
打印结果6

OC:
NSString *stringValue = @"abc李";
NSLog(@"%tu", stringValue.length);
打印结果4, 以UTF16计算
*/

var stringValue = "abc李"
print(stringValue.lengthOfBytesUsingEncoding(NSUTF8StringEncoding))
// 打印结果6, 和C语言一样计算字节数

字符串拼接

char str1[] = "abc";

NSMutableString *str1 = [NSMutableString stringWithString:@"abc"];

[str1 appendString:str2];

运算符

算术运算符：除了取模，其他跟OC一样，包括优先级。

注意:Swift是安全严格的编程语言, 会在编译时候检查是否溢出, 但是只会检查字面量而不会检查变量, 所以在Swift中一定要注意隐式溢出

可以检测
var num1:UInt8 = 255 + 1;
无法检测
var num1:UInt8 = 255
var num2:UInt8 = 250
var result = num1 + num2
println(result)

遇到这种问题可以利用溢出运算符解决该问题:http://www.yiibai.com/swift/overflow_operators.html

溢出运算符：&+ &- &* &/ &%

取模：求余”比”取模”

/*
取模 %
OC: 只能对整数取模
NSLog(@"%tu", 10 % 3);

Swift: 支持小数取模
*/
print(10 % 3.5)

注意: 
 1.Swift刚出来时的写法是 a..b
 2.区间只能用于整数, 写小数会有问题

for i in 1...5
