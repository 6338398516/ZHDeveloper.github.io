Swift枚举

发表于 2016-05-09   |   分类于 枚举   |     |   阅读次数

枚举:

Swift中的枚举比OC中的枚举强大，因为Swift中的枚举是一等类型，它可以想类和结构体一样增加属性和方法。

格式：

enum EnumName {
    case 枚举值
}

enum operation {
    case Add,Sub,Mul,Div
}

//可以使用常亮或者变量来接受枚举值
let operation:Operation = .Add
// 注意: 如果变量或常量没有指定类型, 那么前面必须加上该值属于哪个枚举类型
let operation1 = Operation.Div

枚举与Swift结合
// 注意: 如果case中包含了所有的值, 可以不写default. 如果case中没有包含枚举中所有的值, 必须写default
switch operation {
    case Operation.Add:
        print("1")
    case Operation.Sub:
        print("1")
    case Operation.Mul:
        print("1")
    case Operation.Div:
        print("1")
}

枚举中的原始值:OC中枚举的本质就是整数,所以OC中的枚举是有原始值的,默认是从0开始而Swift中的枚举默认是没有原始值的, 但是可以在定义时告诉系统让枚举有原始值。

//和OC中的枚举一样, 也可以指定原始值, 后面的值默认+1
//Swift中的枚举除了可以指定整形以外还可以指定其它类型, 但是如果指定其它类型, 必须给所有枚举值赋值, 因为不能自动递增
enum Operation1:Int {
    case Add = 1 ,Sub,Mul,Div
}

// rawValue代表将枚举值转换为原始值, 注意老版本中转换为原始值的方法名叫toRaw
print(Operation1.Add.rawValue)

通过原始值创建枚举值

  1. 原始值区分大小写。
  2. 返回的值是一个可选类型。以为原始值对应的枚举值不一定存在。
  3. 老版本中为fromRaw(“add”)

eg:

let enumValue = Operation1(rawValue: "1")

枚举相关值:

可以让枚举对应的原始值不是唯一，而是一个变量。

enum lineSegmentDescriptor{
    case StartAndEndPattern(start:Double, end:Double)
    case StartAndLengthPattern(start: Double, length:Double)
}

var lsd = lineSegmentDescriptor.StartAndLengthPattern(start: 0.0, length: 100.0)
lsd = lineSegmentDescriptor.StartAndEndPattern(start: 0.0, end: 50.0)
// 提取关联值优化写法
switch lsd
{
        case let .StartAndEndPattern(s, e):
            print("start = \(s) end = \(e)")
        case .StartAndLengthPattern(let s, let l):
            print("start = \(s) lenght = \(l)")
}

Swift闭包

闭包:

  1. 函数是闭包的一种特殊。
  2. 闭包表达式（匿名闭包）- 能捕获上下文中闭包的值。

闭包的语法:

  1. in关键字是便于区分返回值和执行语句。
  2. 闭包表达式的类型和函数的类型一样，是参数上加返回值。也就是in之前的部分。
  3. 如果闭包是最后一个参数, 可以直接将闭包写到参数列表后面, 这样可以提高阅读性. 称之为尾随闭包

eg:

//完整写法
let say:(String)->String = {
    (name:String) -> String in
    return "hello \(name)"
}

//没有返回值的写法
let say:(String)->Void = {
    (name:String) in
    print("hello \(name)")
}

//无参无返回值
let say = {
    print("hello")
}

func bubbleSort(inout array:[Int], cmp: (Int, Int) -> Int)
{
    let count = array.count;
    for var i = 1; i < count; i++
    {
        for var j = 0; j < (count - i); j++
        {
            if cmp(array[j], array[j + 1]) == -1
            {
                let temp = array[j]
                array[j] = array[j + 1]
                array[j + 1] = temp
            }
        }
    }
}

// 闭包作为参数传递
bubbleSort(&arr, cmp: {
    (a: Int, b: Int) -> Int in
    if a > b{
        return 1;
    }else if a < b
    {
        return -1;
    }else
    {
        return 0;
    }
})

闭包表达式的优化:

  1. 类型优化, 由于函数中已经声明了闭包参数的类型, 所以传入的实参可以不用写类型
  2. 返回值优化, 同理由于函数中已经声明了闭包的返回值类型,所以传入的实参可以不用写类型
  3. 参数优化, swift可以使用$索引的方式来访问闭包的参数, 默认从0开始
  4. 如果只有一条语句可以省略return

eg:

bubbleSort(&arr){
//    (a , b) -> Int in
//    (a , b) in
    if $0 > $1{
        return 1;
    }else if $0 < $1
    {
        return -1;
    }else
    {
        return 0;
    }
}

Swift函数

函数:完成特定功能的代码块。给函数定义一个有意义的名字，在合适的时候调用。

定义:
func 函数名称(参数名:参数类型, 参数名:参数类型…) -> 函数返回值 {函数实现部分}

//无返回值
func say() -> Void{
    print("hello")
}

func say() {
    print("hello")
}

//有参无返回值
func sayWith(name:String) {
    print("hello \(name)")
}

//无参有返回值
func personInfo() -> (String,Int) {
    return ("zhangsan",23)
}

//有参有返回值
func info(name:String, age:Int) -> String
{
    return "name = \(name), age = \(age)"
}

//函数嵌套
func bubbleSort(inout array:[Int]) -> Void {

    //定义一个交换函数
    func swap(inout a:Int,inout b:Int)
    {
        let temp = a
        a = b
        b = temp
    }

    let count = array.count

    for i in 1..<count {
        for j in 0 ..< (count-i) {
            if array[j]>array[j+1]
            {
                swap(&array[j], b: &array[j+1])
            }
        }
    }

}

//要使用inout必须是变量，因为inout会改变值
var array = [54,43,6,63,23,1,52]
bubbleSort(&array)

函数参数

  1. 函数的第一个参数默认是内部参数，之后的参数即是内部参数也是外部参数。
  2. 自定义外部参数，在内部参数前加外部参数和空格
  3. Swift2.0过时,在参数前面加上#相当于该参数即是内部参数, 也是外部参数
  4. 参数默认值，在定义函数的时候给某个参数赋值, 当外部调用没有传递该参数时会自动使用默认值
  5. 在其它语言中默认参数智能出现在参数列表的最后面, 但是在Swift中可以出现在任何位置
  6. inout参数, 如果想在函数中修改外界传入的参数, 传递参数本身而不是参数的值
  7. …表示函数的参数有多个，相当于数组。变参必须放到函数列表的最后。

eg:

dividend和divisor都是外部参数，a和b是内部参数
func divisionOpertaion(dividend a: Double, divisor b:Double) -> Double{
    return a / b
}

//参数默认值的使用
func calculateWith(number1 a:Int,andNumber b:Int = 10) -> Int {
    return a+b
}

let a = calculateWith(number1: 8)

函数类型

函数的类型是由参数类型和返回值的类型组成。
可以使用函数类型来定义常亮和变量。

//函数的类型为(Double,Double)->Double
func divisionOpertaion(dividend a: Double, divisor b:Double) -> Double{
    return a / b
}

//函数类型作为参数，在函数内部调用
func calFunc(a: Int, b: Int ,Option:(Int, Int) -> Int) ->Int
{
    return Option(a, b)
}

//函数类型作为返回值
func max(a:Int,b:Int) -> Int {
    return a>b ? a:b
}

func min(a:Int,b:Int) -> Int {
    return a<b ? a:b
}

func getMax(getMax:Bool) -> (Int,Int)->Int {
    return getMax ? max : min
}

let choose = getMax(true)

if分支

if 条件表达式 {指令} if 条件表达式 {指令} else{指令}

  1. if后的圆括号可以省略
  2. 只能以bool作为条件语句
  3. 如果只有条指令if后面的大括号不可以省略

eg:

var i:Int = 0
var sum:Int = 0
while (i <= 10)
{
    sum = i++
}
print("\(sum)")

var i1:Int = 0
var sum1:Int = 0
while i1 <= 10
{
    sum1 = i1++
}
print("\(sum1)")

while循环

  1. while后的圆括号可以省略
  2. 只能以bool作为条件语句
  3. 如果只有条指令while后面的大括号不可以省略

eg:

var i:Int = 0
var sum:Int = 0
while (i <= 10)
{
    sum = i++
}
print("\(sum)")

break、continue

  1. break结束当前循环。
  2. continue忽略continue后面的语句。

switch

特点：

  1. 默认不可以穿透，使用关键字fallthrough使语句穿透
  2. 可以不写break
  3. 必须有default
  4. 在case中定义变量可以不用加大括号
  5. 可以和区间和元组匹配
  6. switch的结构相对于if(){}else if(){}else{}

eg.

var rank1 = "A"
switch rank1{
    case "A", "B": // 注意OC不能这样写
        print("优")
    case "C":
        print("差")
    default:
        print("没有评级")
}
//两个实现的效果是一样的
switch (rank) {
    case "A":
        fallthrough
    case "B":
        NSLog("优");
    case "C":
        NSLog("差");
    default:
        NSLog("没有评级");
}

//与区间匹配
let aNum = 10

switch aNum {
    case 0...10:
        print(0)
    case 10...20:
        print(1)
    case 20...30:
        print(2)
    case 30...40:
        print(3)
    default:
        print(aNum)
}
//区间与元组搭配
let aNum = (89,97)

switch aNum {
    case (0..<60,0..<60):
        print("不及格")
    case (60..<70,60..<70):
        print("差")
    case (70..<80,60..<80):
        print("良")
    case (80...100,80...100):
        print("优秀")
    default:
        print(aNum)
}

//值绑定
var point = (1, 10)
switch point{
    case (var x, 10): // 会将point中X的值赋值给X
        print("x= \(x)")
    case (var x, var y): // 会将point中XY的值赋值给XY
        print("x= \(x) y= \(y)")
    case var( x, y):
        print("x= \(x) y= \(y)")
    default:
        print("Other")
}

//根据条件绑定
var point = (100, 10)
switch point{
    // 只有where后面的条件表达式为真才赋值并执行case后的语句
    case var(x, y) where x > y: 
        print("x= \(x) y= \(y)")
    default:
        print("Other")
}
